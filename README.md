# Improved Scroll

Improved Resource Scroll mod for Minecraft 1.20.1.

## Description
This mod improves the resource scroll functionality, specifically designed to work with MineColonies.

## Requirements
- Minecraft 1.20.1
- Minecraft Forge
- MineColonies

## Installation
1. Install Minecraft Forge for 1.20.1.
2. Install MineColonies.
3. Drop the `improved_scroll-1.0-SNAPSHOT.jar` into your `mods` folder.

## Build
To build the project, run:
```bash
./gradlew build
```
The output jar will be in `build/libs/`.
