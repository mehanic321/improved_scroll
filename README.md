# Improved Scroll

Improved Resource Scroll mod for Minecraft 1.20.1.

## Description
This mod improves the resource scroll functionality, specifically designed to work with MineColonies. It allows you to easily fetch requested resources from any colony building's containers directly into your inventory.

## How it works
1.  **Link the Scroll:** Right-click on a **Builder's Hut** block to link the scroll to that specific builder.
2.  **Check Requests:** Right-click anywhere (without Shift) to see the list of resources the builder needs.
3.  **Gather Resources:** Go to *any* Colony building (e.g., a Warehouse, Forester, Miner). Hold **Shift** and **Right-click**. The scroll will automatically search all containers in that building for items the linked builder needs and transfer them to your inventory.

## Requirements
- Minecraft 1.20.1
- Minecraft Forge
- MineColonies

## Installation
1. Install Minecraft Forge for 1.20.1.
2. Install MineColonies.
3. Drop the `improved_scroll-1.0-SNAPSHOT.jar` into your `mods` folder.

## Build
To build the project, run:
```bash
./gradlew build
```
The output jar will be in `build/libs/`.

# Improved Scroll (Русский)

Мод, улучшающий функционал ресурсного свитка для MineColonies на версии 1.20.1.

## Описание
Этот мод улучшает работу ресурсного свитка, позволяя легко собирать запрошенные ресурсы из контейнеров любого здания колонии прямо в ваш инвентарь.

## Как это работает
1.  **Привязка свитка:** Нажмите **ПКМ** по блоку **Хижины Строителя**, чтобы привязать свиток к этому конкретному строителю.
2.  **Просмотр запросов:** Нажмите **ПКМ** в любом месте (без Shift), чтобы открыть список ресурсов, необходимых строителю.
3.  **Сбор ресурсов:** Зайдите в *любое* здание колонии (например, Склад, Лесоруб, Шахтер). Зажмите **Shift** и нажмите **ПКМ**. Свиток автоматически проверит все контейнеры в этом здании на наличие предметов, нужных привязанному строителю, и переместит их в ваш инвентарь.

## Требования
- Minecraft 1.20.1
- Minecraft Forge
- MineColonies

## Установка
1. Установите Minecraft Forge для 1.20.1.
2. Установите MineColonies.
3. Поместите файл `improved_scroll-1.0-SNAPSHOT.jar` в папку `mods`.

## Сборка
Для сборки проекта выполните команду:
```bash
./gradlew build
```
Готовый jar-файл будет находиться в папке `build/libs/`.
